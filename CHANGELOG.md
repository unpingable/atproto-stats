# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-02-27

- Initial local-only `bsky-noise` release.
- Added auth/sync/report CLI with SQLite caching and incremental refresh.
- Added static HTML + JSON reporting with sorting/search/focus mode/dark mode.
- Added action metrics: attention share, prior-window deltas, what-if mute impact.
- Added run health receipts and resilient sync modes (degraded/auto-degraded).
- Added orbit candidate discovery/scoring/shortlist workflow.
- Added project docs and dual license (MIT OR Apache-2.0).
